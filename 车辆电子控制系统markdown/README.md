## 技术趋势

### 从分布式到中央计算

```
E/E架构演进路线:

传统架构 (2010前)          域集中架构 (2015-2025)        中央计算架构 (2025+)
┌─────────────┐            ┌─────────────┐               ┌─────────────┐
│  发动机ECU  │            │   动力域    │               │             │
├─────────────┤            │   (PDCU)    │               │   中央计算  │
│  变速箱ECU  │            ├─────────────┤               │   大脑      │
├─────────────┤            │   底盘域    │               │   (CDC)     │
│  ABS/ESP    │  ──────▶   │   (CDC)     │  ─────────▶   ├─────────────┤
├─────────────┤            ├─────────────┤               │   区域控制器 │
│   仪表ECU   │            │   座舱域    │               │   (ZCU)     │
├─────────────┤            │   (CDC)     │               └─────────────┘
│   娱乐ECU   │            ├─────────────┤                      │
├─────────────┤            │  自动驾驶域  │                      ▼
│   空调ECU   │            │   (ADC)     │               ┌─────────────┐
└─────────────┘            └─────────────┘               │   传感器    │
   70-100+ ECU                   4-6域控                 │   执行器    │
                                                          └─────────────┘
```

### AI算力需求增长

| 自动驾驶级别 | 算力需求(TOPS) | 典型芯片 | 量产时间 |
|-------------|---------------|---------|---------|
| L2+ | 10-30 | Mobileye EyeQ4 | 2018 |
| L2++ | 30-100 | NVIDIA Xavier | 2020 |
| L3 | 100-500 | Orin-X (254TOPS) | 2022 |
| L4 | 500-2000 | Thor (2000TOPS) | 2025 |
| L5 | 2000+ | 自研芯片 | 待定 |

### 通信带宽演进

| 系统 | 传统CAN | CAN-FD | Ethernet | 需求驱动 |
|-----|---------|--------|----------|---------|
| 动力系统 | 500kbps | 2-5Mbps | - | 实时控制 |
| 底盘系统 | 500kbps | 2Mbps | - | 安全关键 |
| ADAS | - | - | 100Mbps-1Gbps | 摄像头数据 |
| 信息娱乐 | - | - | 1Gbps+ | 4K/8K视频 |
| 自动驾驶 | - | - | 10Gbps+ | 多路激光雷达 |
