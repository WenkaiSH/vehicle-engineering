### 2.3 摄像头感知算法

| 任务 | 算法类型 | 输出 | 算力需求 |
|-----|---------|------|---------|
| **目标检测** | YOLO/SSD/Faster R-CNN | 2D包围框+类别 | 中等 |
| **语义分割** | U-Net/DeepLab | 像素级类别 | 高 |
| **深度估计** | MonoDepth/BEVDepth | 深度图 | 中等 |
| **车道线检测** | LaneNet/PINet | 车道线方程 | 低 |
| **交通标志识别** | CNN分类 | 标志类型 | 低 |
| **红绿灯识别** | CNN检测+分类 | 灯状态 | 低 |

### 2.4 摄像头优势与局限

```
摄像头特性:

优势 (Strengths):
├─ ✓ 分辨率高，纹理信息丰富
├─ ✓ 能识别颜色、文字、标志
├─ ✓ 成本低，技术成熟
├─ ✓ 体积小，易于集成
└─ ✓ 与人类视觉一致，可解释性强

局限 (Weaknesses):
├─ ✗ 受光照影响大 (强光/黑夜/逆光)
├─ ✗ 雨雾天气性能下降
├─ ✗ 无法直接获取深度信息 (单目)
├─ ✗ 对纹理少的物体检测困难 (白墙/天空)
└─ ✗ 需要大量训练数据

弥补方案:
├─ HDR技术应对光照变化
├─ 红外补光应对夜间
├─ 多目立体视觉获取深度
├─ 与雷达/激光雷达融合
└─ 数据闭环持续优化
```

## 3. 激光雷达 (LiDAR)

### 3.1 激光雷达原理与分类

```
激光雷达工作原理:

发射端                           接收端
┌─────────┐                     ┌─────────┐
│ 激光器   │───激光脉冲────────▶│  接收器  │
│ (905nm  │    ↓ 目标反射      │ (APD/   │
│  1550nm)│◀───────反射光──────│  SPAD)  │
└────┬────┘                     └────┬────┘
     │                                │
     │      ┌──────────────┐         │
     └─────▶│  ToF测距      │◀────────┘
            │  d = c×t/2   │
            └──────────────┘

测距原理: Time of Flight (飞行时间)
├─ 发射激光脉冲
├─ 记录发射时间t1
├─ 接收反射光，记录时间t2
└─ 距离 d = c × (t2-t1) / 2

扫描方式分类:
┌─────────────────────────────────────────────────────────┐
│ 机械式         MEMS        固态-OPA       固态-Flash   │
│ ┌─────┐      ┌─────┐      ┌─────┐       ┌─────┐       │
│ │旋转 │      │微镜 │      │相控阵│       │面阵 │       │
│ │马达 │      │偏转 │      │扫描 │       │一次 │       │
│ └─────┘      └─────┘      └─────┘       └─────┘       │
│  360°         120°         120°          60°          │
│  成本高       中成本       低成本        短距离        │
│  车规难       主流方案     待量产        补盲用        │
└─────────────────────────────────────────────────────────┘
```

## 4. 毫米波雷达

### 4.1 毫米波雷达原理

```
毫米波雷达工作原理:

┌─────────────────────────────────────────────────────────┐
│                  毫米波雷达系统                          │
├─────────────────────────────────────────────────────────┤
│  发射信号: FMCW (调频连续波)                              │
│                                                         │
│  f    ┌──────┐                                          │
│   ▲   │      │     发射频率随时间线性变化                 │
│   │  /        \                                         │
│   │ /          \        ┌──────────────────┐            │
│   │/            \──────▶│     目标反射      │            │
│   └─────────────────▶   └────────┬─────────┘            │
│         时间                     │                      │
│                                  ▼                      │
│                           ┌──────────────┐             │
│                           │   混频器     │             │
│                           │  发射-接收    │             │
│                           │   = 差频信号  │             │
│                           └──────┬───────┘             │
│                                  │                      │
│                                  ▼                      │
│                           ┌──────────────┐             │
│                           │  信号处理    │             │
│                           │  FFT→距离    │             │
│                           │  多普勒→速度  │             │
│                           │  相位→角度    │             │
│                           └──────────────┘             │
└─────────────────────────────────────────────────────────┘

关键公式:
· 距离: R = c × τ / 2  (τ为往返时间)
· 速度: v = λ × fd / 2  (fd为多普勒频移)
· 角度: θ = arcsin(λ × Δφ / 2πd) (Δφ为相位差)
```
