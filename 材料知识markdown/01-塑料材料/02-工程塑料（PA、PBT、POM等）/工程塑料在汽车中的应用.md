# 工程塑料在汽车中的应用

> PA、PBT、POM 等工程塑料的汽车应用详解

---

## 概述

工程塑料具有优异的机械性能、耐热性和耐化学性，在汽车中主要用于承受载荷、高温环境和精密功能部件。

---

## 聚酰胺（PA / 尼龙）

### 基本信息

| 类型 | 熔点 | 热变形温度 | 价格区间 |
|------|------|-----------|---------|
| PA6 | 220°C | 65-85°C | 25-35 元/kg |
| PA66 | 265°C | 70-100°C | 30-45 元/kg |
| PA46 | 295°C | 150°C+ | 80-120 元/kg |
| PPA | 310°C | 200°C+ | 150-250 元/kg |

### 汽车应用

| 应用部位 | 典型零件 | 材料选择 |
|----------|---------|---------|
| 动力系统 | 进气歧管、气门室罩盖 | PA6+GF30、PA66+GF30 |
| 冷却系统 | 散热器水箱、风扇护罩 | PA66+GF |
| 电气系统 | 连接器、扎带、线束护套 | PA66纯料或阻燃级 |
| 座椅系统 | 调角器齿盘、滑轨滑块 | PA66+GF50 |
| 电机系统 | 电机壳体、控制器壳体 | 高温尼龙PPA |

### 改性方向

| 改性类型 | 添加物 | 效果 |
|----------|-------|------|
| 玻纤增强 | 玻璃纤维10-50% | 强度↑、刚度↑、尺寸稳定性↑ |
| 阻燃改性 | 无卤阻燃剂 | 阻燃V0级 |
| 耐热改性 | 矿物填充 | 热变形温度↑ |
| 耐磨改性 | PTFE、二硫化钼 | 摩擦系数↓ |

## 聚对苯二甲酸丁二醇酯（PBT）

### 基本信息

| 属性 | 数值 |
|------|------|
| 密度 | 1.31 g/cm³ |
| 熔点 | 225°C |
| 热变形温度 | 60-150°C（取决于改性） |
| 价格区间 | 35-70 元/kg |

### 汽车应用

| 应用部位 | 典型零件 | 材料要求 |
|----------|---------|---------|
| 电气系统 | 连接器、开关、继电器 | 阻燃V0、尺寸稳定 |
| 传感器 | 外壳、支架 | 耐化学性、绝缘性 |
| 车灯 | 灯座、反射镜 | 耐热、耐候 |

### 技术趋势

- **无卤阻燃**：满足环保法规要求
- **激光焊接**：实现精密连接
- **高流动性**：薄壁化、小型化

## 聚甲醛（POM）

### 基本信息

| 属性 | 数值 |
|------|------|
| 密度 | 1.41 g/cm³ |
| 熔点 | 165-175°C |
| 价格区间 | 40-80 元/kg |

### 特点

- 优异的自润滑性
- 低摩擦系数
- 高尺寸精度
- 良好的耐疲劳性

### 汽车应用

| 应用部位 | 典型零件 | 原因 |
|----------|---------|------|
| 座椅系统 | 滑轨滚轮、齿轮 | 低摩擦、耐磨 |
| 门锁系统 | 锁芯、齿轮 | 精密、耐久 |
| 安全带 | 卷收器齿轮 | 耐疲劳 |
| 空调系统 | 风门执行机构 | 低摩擦、静音 |
| 隐藏式出风口 | 传动机构 | 精密、低摩擦 |

### 改性类型

| 改性类型 | 效果 | 应用 |
|----------|------|------|
| 低摩擦级 | 摩擦系数进一步降低 | 高负荷滑动部件 |
| 耐磨级 | 耐磨性提升 | 高磨损环境 |
| 抗UV级 | 耐候性提升 | 外饰件 |

## 性能对比

| 性能 | PA66 | PBT | POM |
|------|------|----|----|
| 密度(g/cm³) | 1.14 | 1.31 | 1.41 |
| 拉伸强度(MPa) | 80-90 | 50-60 | 60-70 |
| 弯曲模量(GPa) | 2.5-3.0 | 2.2-2.6 | 2.5-2.8 |
| 热变形温度(°C) | 70-250* | 60-210* | 100-160 |
| 摩擦系数 | 0.3-0.5 | 0.4-0.6 | 0.15-0.35 |
| 耐油性 | 优 | 良 | 优 |

*取决于玻纤含量

---

## 选型建议

| 应用需求 | 推荐材料 |
|----------|---------|
| 高强度结构件 | PA66+GF30/50 |
| 高温环境（>150°C） | 高温尼龙PPA/PA46 |
| 精密齿轮/滑动 | POM |
| 电气连接器 | PBT阻燃级、PA66阻燃级 |
| 耐油部件 | PA、POM |
