# 刚度与强度分析

## 概述

**刚度与强度**是白车身最核心的结构性能指标。刚度决定车辆的操控稳定性和NVH性能，强度决定车辆的承载能力和安全性。CAE仿真和试验验证是评估这两项指标的主要手段。

---

## 刚度分析

### 弯曲刚度

#### 定义

车身在垂直载荷作用下的抗变形能力。

**计算公式**：
```
弯曲刚度 = 载荷 / 最大挠度
单位：N/mm
```

#### 试验方法

**四点弯曲试验**：
```
        加载点1          加载点2
           ↓                ↓
    ───────┴────────────────┴───────
           ↑                    ↑
         支撑点1              支撑点2
         
    测量：中部位移
```

**标准**：
- 中型轿车目标：≥20,000 N/mm
- 豪华车目标：≥25,000 N/mm
- 电动车目标：≥25,000 N/mm（考虑电池重量）

#### 仿真方法

**边界条件**：
- 约束：前后悬置点
- 载荷：座椅位置施加垂直力

**评价指标**：
- 整体弯曲刚度
- 地板局部刚度
- 变形模式

### 扭转刚度

#### 定义

车身在扭矩作用下的抗扭转变形能力。

**计算公式**：
```
扭转刚度 = 扭矩 / 扭转角
单位：N·m/deg
```

#### 试验方法

**扭转试验**：
```
    前悬置点固定
    ↓
    ┌─────────────────────┐
    │                     │ ← 后悬置点施加扭矩
    │      车身           │
    │                     │
    └─────────────────────┘
    
    测量：前后端相对转角
```

**标准**：
| 车型 | 目标值 |
|------|--------|
| 紧凑型轿车 | 20,000-25,000 |
| 中型轿车 | 25,000-30,000 |
| 豪华车 | 30,000-40,000 |
| 电动车（CTB） | 35,000-45,000 |

#### 仿真方法

**边界条件**：
- 前悬置点：全约束
- 后悬置点：施加扭矩

**影响因素**：
- 地板结构（最重要）
- 侧围与地板连接
- 顶盖刚度
- 前后舱刚度

### 刚度提升策略

#### 地板刚度提升

| 措施 | 效果 | 代价 |
|------|------|------|
| **增加横梁** | 弯曲刚度↑↑ | 重量↑ |
| **增加纵梁截面** | 扭转刚度↑↑ | 重量↑↑ |
| **横梁与纵梁连接加强** | 扭转刚度↑ | 工艺复杂 |
| **中央通道加强** | 扭转刚度↑ | 空间占用 |

#### 电动车刚度优势

**CTB/CTC技术**：
- 电池包参与承载
- 扭转刚度提升50%+
- 比亚迪海豹：40,500 N·m/deg

## 强度分析

### 静强度分析

#### 工况定义

| 工况 | 载荷 | 评价标准 |
|------|------|----------|
| **制动** | 1.0g减速度 | 应力<屈服强度 |
| **转弯** | 0.8g侧向加速度 | 应力<屈服强度 |
| **颠簸** | 3g垂直加速度 | 应力<屈服强度 |
| **组合工况** | 上述组合 | 应力<屈服强度 |

#### 安全系数

```
安全系数 = 屈服强度 / 工作应力

一般要求：≥1.5
关键部位：≥2.0
```

### 疲劳强度分析

#### 疲劳失效机理

**三阶段**：
1. **裂纹萌生**：应力集中处产生微裂纹
2. **裂纹扩展**：循环载荷下裂纹扩展
3. **最终断裂**：剩余截面不足以承载

#### 分析方法

**名义应力法**：
- S-N曲线（应力-寿命曲线）
- Miner累积损伤理论
- 适用于高周疲劳（>10⁴次）

**局部应力应变法**：
- 考虑缺口效应
- 适用于低周疲劳

**断裂力学法**：
- 考虑初始缺陷
- 预测裂纹扩展寿命

#### 载荷谱

**道路载荷谱**：
- 试验场测量
- 典型路面：扭曲路、搓板路、坑洼路
- 等效目标：20万公里

### 碰撞强度

见「电动车碰撞安全」章节

## 刚度与强度优化

### 拓扑优化

**目标**：寻找最优材料分布

**约束**：
- 刚度约束（弯曲、扭转）
- 重量约束
- 制造约束

**应用**：
- 加强筋布置
- 减重孔位置
- 材料分布

### 尺寸优化

**变量**：
- 板料厚度（0.6-2.0mm）
- 梁截面尺寸
- 加强板数量

**目标函数**：
- 重量最小
- 满足刚度/强度约束

**优化结果示例**：
| 部位 | 原厚度 | 优化后 | 变化 |
|------|--------|--------|------|
| 地板 | 0.8mm | 0.7mm | -12% |
| 纵梁 | 1.6mm | 1.8mm | +12% |
| 横梁 | 1.2mm | 1.0mm | -17% |
| **总重量** | - | - | -5% |

### 形状优化

**变量**：
- 截面形状
- 加强筋形状
- 过渡圆角

**应用**：
- 纵梁截面优化
- 吸能盒诱导槽设计
- 接头形状优化

## 试验验证

### 刚度试验

#### 弯曲刚度试验

**设备**：
- 万能试验机
- 位移传感器
- 数据采集系统

**步骤**：
1. 车身固定在试验台
2. 在座椅位置施加垂直载荷
3. 测量位移
4. 计算刚度

#### 扭转刚度试验

**设备**：
- 扭转试验台
- 扭矩传感器
- 角度传感器

**步骤**：
1. 前悬置点固定
2. 后悬置点施加扭矩
3. 测量扭转角
4. 计算刚度

### 强度试验

#### 静强度试验

**方法**：
- 在关键部位粘贴应变片
- 施加设计载荷
- 测量应力分布
- 与仿真结果对比

#### 疲劳试验

**方法**：
- 道路模拟试验机
- 加载实测载荷谱
- 监测裂纹萌生
- 记录失效里程

## 刚度与强度平衡

### 设计矛盾

| 提高刚度/强度 | 负面影响 |
|--------------|----------|
| 增加板厚 | 重量增加 |
| 增加加强筋 | 成本增加，工艺复杂 |
| 使用高强钢 | 成型困难，回弹大 |
| 增加焊点 | 变形增加 |

### 优化策略

**多目标优化**：
```
最小化：重量
约束条件：
  - 弯曲刚度 ≥ 目标值
  - 扭转刚度 ≥ 目标值
  - 最大应力 ≤ 许用应力
  - 成本 ≤ 预算
```

**材料优化**：
- 高应力区：高强钢
- 低应力区：减薄或减重孔
- 碰撞区：梯度强度设计

## 电动车特殊考虑

### 电池包承载

**影响**：
- 地板载荷增加300-600kg
- 局部刚度要求提高
- 连接点强度要求提高

**对策**：
- 地板横梁加强
- 电池包边框与车身一体化
- 连接点增加

### 一体化压铸

**优势**：
- 刚度提升15-20%
- 连接点减少，应力集中降低
- 整体性好

**考虑**：
- 局部损伤影响大
- 需要足够的韧性

---

## 总结

刚度与强度是白车身设计的核心指标，需要在性能、重量、成本之间寻求最佳平衡。通过CAE仿真优化和试验验证，可以在设计阶段确保产品满足目标要求。电动车由于电池包重量，对刚度和强度提出了更高要求，CTB/CTC技术为此提供了有效的解决方案。
