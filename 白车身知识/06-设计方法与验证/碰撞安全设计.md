# 碰撞安全设计

## 概述

**碰撞安全设计**是白车身设计的核心目标之一，通过合理的结构设计和材料选择，在碰撞时保护乘员舱完整性，吸收碰撞能量，降低乘员伤害。电动车还需要额外考虑电池保护。

---

## 碰撞安全理念

### 安全设计原则

```
碰撞安全设计层次：

第1层：避免碰撞（主动安全）
    ↓
第2层：减轻碰撞（主动安全）
    ↓
第3层：保护乘员（被动安全） ← 白车身设计核心
    ↓
第4层：碰撞后安全（救援）
```

### 白车身被动安全目标

1. **乘员舱完整性**：碰撞时不发生侵入
2. **能量吸收**：有效吸收碰撞动能
3. **约束系统配合**：为气囊/安全带提供支撑
4. **逃生空间**：碰撞后车门可开启

## 碰撞类型与设计

### 正面碰撞

#### 传力路径设计

```
正面碰撞能量传递：

前保险杠 → 前横梁 → 吸能盒（压溃） → 前纵梁（压溃）
                                              ↓
                        乘员舱（刚性保护，不变形） ← 门槛梁
                                              ↓
                                         后纵梁 → 后保险杠
```

#### 前舱分区设计

| 区域 | 功能 | 设计要求 |
|------|------|----------|
| **前保险杠-前横梁** | 低速碰撞保护 | 可更换，低成本 |
| **吸能盒** | 中速碰撞吸能 | 可控压溃 |
| **前纵梁前段** | 高速碰撞吸能 | 渐进压溃 |
| **前纵梁后段** | 载荷传递 | 高刚度，不压溃 |
| **前围板** | 乘员舱保护 | 高刚度，防侵入 |

#### 吸能结构设计

**压溃诱导设计**：
```
纵梁压溃诱导槽：

截面A（完整）：      截面B（诱导槽）：
┌────────┐          ┌────┬────┐
│        │          │    ↓    │  ← 诱导槽
│        │    →     ├────┴────┤
│        │          │         │
└────────┘          └─────────┘

压溃顺序：从诱导槽开始，逐步压溃
```

**材料选择**：
- 前段：DP590/DP780（高强度，可压溃）
- 后段：热成型钢（高刚度，不压溃）

### 侧面碰撞

#### 传力路径设计

```
侧面碰撞能量传递：

侧向冲击 → 车门 → B柱 → 门槛梁 + 地板横梁 → 非碰撞侧
              ↓
         顶盖横梁（分担载荷）
```

#### 侧面保护结构

| 结构 | 功能 | 材料 |
|------|------|------|
| **车门防撞梁** | 第一道防线 | 高强钢/铝合金 |
| **B柱加强板** | 核心保护 | 热成型钢1500MPa |
| **门槛梁** | 主要承载 | 热成型钢/铝型材 |
| **地板横梁** | 载荷分散 | 高强钢 |
| **顶盖横梁** | 上部支撑 | 高强钢 |

#### B柱设计

**多层板设计**：
```
B柱截面（从上到下）：

外层：0.7mm 深冲钢（外观）
加强层：1.6mm 热成型钢（1500MPa）
内层：0.8mm 高强钢（结构）
```

**厚度梯度**：
- 上段：较厚（支撑车顶）
- 中段：最厚（腰部保护）
- 下段：较薄（铰链/锁扣区域）

### 追尾碰撞

#### 传力路径设计

```
追尾碰撞能量传递：

后保险杠 → 后横梁 → 后吸能盒 → 后纵梁（压溃）
                                              ↓
                                         地板纵梁
                                              ↓
                        乘员舱（刚性保护，不变形）
```

#### 后舱设计

**燃油车尾追**：
- 燃油箱保护
- 燃油管路完整性

**电动车追尾**：
- 电池包后部保护
- 后悬置脱落保护

### 翻滚保护

#### 车顶强度

**要求**：
- 承受3.5-4倍车重
- 变形量<127mm

**设计**：
- A/B/C柱加强
- 顶盖纵梁
- 侧围上边梁

## 碰撞仿真技术

### 显式动力学分析

**原理**：
```
显式积分：
Δt很小（10⁻⁶s级）
考虑惯性效应
适合大变形、瞬态问题
```

**软件**：
- LS-DYNA
- Abaqus/Explicit
- PAM-CRASH

### 仿真流程

```
1. 几何模型（白车身+内外饰）
2. 网格划分（整车约500万单元）
3. 材料定义（含失效模型）
4. 接触定义（自接触、对接触）
5. 边界条件（初速度、壁障）
6. 求解（数小时-数天）
7. 结果分析（变形、加速度、侵入量）
```

### 壁障模型

| 测试类型 | 壁障类型 | 壁障特性 |
|----------|----------|----------|
| **100%正面** | 刚性墙 | 无变形 |
| **40%偏置** | 可变形壁障 | 模拟对向车 |
| **侧面** | 可移动壁障 | 950kg，模拟侧面撞击 |
| **柱碰** | 刚性柱 | φ254mm |

### 假人模型

| 类型 | 用途 |
|------|------|
| **Hybrid III** | 正面碰撞 |
| **SID** | 侧面碰撞 |
| **BioRID** | 追尾碰撞 |
| **THOR** | 新一代假人 |

### 评价指标

**加速度**：
- 头部加速度 < 80g
- 胸部加速度 < 60g

**侵入量**：
- 转向管柱 < 80mm
- A柱 < 100mm
- 踏板 < 100mm

**伤害值**：
- HIC（头部伤害指数）< 700
- Chest Deflection（胸部压缩量）< 63mm
- Femur Force（股骨力）< 10kN

## 碰撞试验验证

### 实车碰撞试验

**正面碰撞**：
- 速度：50-64 km/h
- 壁障：刚性墙/可变形壁障
- 假人：驾驶员+乘员

**侧面碰撞**：
- 速度：50 km/h
- 壁障：可移动壁障
- 假人：驾驶员侧

**柱碰试验**：
- 速度：32-35 km/h
- 壁障：刚性柱
- 电动车新增要求

### 碰撞后检查

| 检查项 | 要求 |
|--------|------|
| **车门开启** | 不借助工具可开启 |
| **燃油泄漏** | 碰撞后5min内≤30g |
| **高压系统** | 无漏电，电压<60V |
| **电池状态** | 无起火、无爆炸 |

## 发展趋势

1. **小偏置碰撞**：加强前舱外侧结构
2. **自动驾驶**：碰撞形态变化，新测试场景
3. **电动车专属**：电池保护测试标准化
4. **行人保护**：机盖、保险杠优化
5. **多车碰撞**：车联网协同避碰

---

## 总结

碰撞安全设计是白车身设计的核心任务，通过合理的传力路径、吸能结构和乘员舱保护，可以在碰撞中最大限度保护乘员。电动车时代增加了电池保护和高压安全的新要求，碰撞安全设计正在向更全面、更严格的方向发展。
