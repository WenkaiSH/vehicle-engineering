# 电池包壳体结构

## 概述

**电池包壳体**是电动汽车动力电池系统的外部保护结构，承载着电芯、模组及相关电气系统，同时提供机械防护、密封防护和热管理功能。在电动车时代，电池包壳体已从单纯的"容器"演变为车身的结构件。

---

## 电池包壳体的功能

### 1. 机械防护

| 防护类型 | 要求 | 实现方式 |
|----------|------|----------|
| **碰撞防护** | 吸收和分散冲击能量 | 边框加强、吸能结构 |
| **振动防护** | 承受道路振动 | 结构强度、固定点设计 |
| **挤压防护** | 承受100kN挤压力 | 边框强度、内部支撑 |
| **穿刺防护** | 防止底部穿刺 | 底板厚度、护板 |

### 2. 环境密封

| 密封要求 | 等级 | 意义 |
|----------|------|------|
| **防尘** | IP6X | 完全防尘 |
| **防水** | IPX7 | 短时浸水保护 |
| **防腐蚀** | 盐雾720h | 延长寿命 |

### 3. 热管理

| 功能 | 实现方式 |
|------|----------|
| **散热** | 液冷板、风道 |
| **保温** | 隔热材料、保温层 |
| **均温** | 导热材料、流道设计 |

### 4. 电气安全

- 绝缘保护
- 高压互锁
- 电位均衡
- 电磁屏蔽

## 电池包壳体结构组成

### 1. 上盖

**功能**：
- 密封电芯舱
- 乘员舱地板（CTB方案）
- 隔热/阻燃

**结构形式**：

| 形式 | 特点 | 材料 | 应用 |
|------|------|------|------|
| **平板式** | 简单，成本低 | 钢板/铝板 | 传统方案 |
| **加强筋式** | 刚度好 | 冲压钢板 | 主流方案 |
| **集成式** | 兼任地板 | 高强钢/铝 | CTB方案 |

**材料选择**：
- 钢板：DC04, HSLA340（厚度0.8-1.2mm）
- 铝板：6016-T4, 5182-O（厚度1.0-1.5mm）
- 复合材料：SMC, CFRP（高端）

### 2. 边框

**功能**：
- 容纳电芯/模组
- 与车身连接
- 侧面碰撞保护

**结构形式**：

```
边框截面演进：

第一代：┌────┐    第二代：┌──┬──┐    第三代：┌──┬──┬──┐
        │    │            │  │  │            │  │  │  │
        └────┘            └──┴──┘            ├──┼──┼──┤
                                              │  │  │  │
                                              └──┴──┴──┘
        单腔体              双腔体              多腔体
```

**材料选择**：

| 材料 | 成型方式 | 特点 | 成本 |
|------|----------|------|------|
| **6061-T6** | 挤压 | 轻量化，易成型 | 中 |
| **6082-T6** | 挤压 | 强度高 | 中 |
| **高强钢** | 辊压 | 成本低，强度高 | 低 |
| **热成型钢** | 热压 | 极高强度 | 高 |

### 3. 底板

**功能**：
- 承载电芯重量
- 底部防护
- 密封（与边框配合）

**结构形式**：

| 形式 | 特点 | 重量 | 刚度 |
|------|------|------|------|
| **单层钢板** | 简单 | 重 | 一般 |
| **单层铝板** | 轻量 | 轻 | 一般 |
| **加强筋式** | 刚度好 | 中等 | 好 |
| **蜂窝夹芯** | 高刚度 | 轻 | 极好 |
| **一体化压铸** | 集成度高 | 中等 | 好 |

**厚度要求**：
- 钢板：1.5-2.5mm
- 铝板：2.0-3.0mm
- 复合材料：根据设计

### 4. 内部结构

#### 模组框架

**功能**：
- 固定电芯/模组
- 提供安装接口
- 保持电芯间距

**材料**：
- 铝合金型材（主流）
- 高强钢板
- 复合材料（探索）

#### 横梁/纵梁

**功能**：
- 内部加强
- 载荷分散
- 模组支撑

**布置方式**：
```
电池包内部梁布置：

纵梁：│  │  │  │  │  （沿车辆纵向）
      ─────────────
横梁：────────────────  （沿车辆横向）
      │  │  │  │  │
      
组合：网格状布置
```

## 电动车电池包壳体特点

### 1. 高刚度需求

**原因**：
- 电池包参与整车承载
- 扭转刚度要求高
- 路面振动传递

**设计措施**：
- 边框多腔体设计
- 底板加强筋
- 与车身多点连接

### 2. 侧面碰撞保护

**传统设计**：
- 门槛梁保护
- 电池包侧面基本无保护

**电动车优化**：
- 门槛梁+电池包边框联合保护
- 电池包边框采用高强材料
- 侧向吸能设计

### 3. 底部防护

**风险**：
- 托底损伤
- 石子冲击
- 涉水腐蚀

**防护措施**：
```
底部防护层次（从外到内）：

第1层：下护板（可更换）
        ↓
第2层：防护涂层（PVC/树脂）
        ↓
第3层：底板（1.5-2.5mm）
        ↓
第4层：液冷板（如有）
        ↓
第5层：绝缘层
        ↓
第6层：电芯
```

## 制造工艺

### 1. 边框制造

**铝合金边框**：
```
铝棒 → 挤压成型 → 时效处理 → 机加工 → 焊接/连接
```

**钢边框**：
```
钢带 → 辊压成型 → 激光切割 → 焊接
```

### 2. 底板制造

**冲压底板**：
```
板材 → 落料 → 拉延 → 修边冲孔 → 翻边
```

**压铸底板**：
```
铝合金熔炼 → 压铸 → 去毛刺 → 机加工 → 热处理
```

### 3. 总成焊接

**焊接顺序**：
1. 边框与底板焊接
2. 内部框架安装
3. 附件焊接（吊耳、接口座）
4. 密封面加工

**连接方式**：
- FSW（搅拌摩擦焊）：铝-铝
- 激光焊：钢-钢、铝-铝
- CMT（冷金属过渡）：钢-铝

### 4. 表面处理

| 工艺 | 作用 | 应用部位 |
|------|------|----------|
| **电泳** | 防腐 | 钢边框、底板 |
| **阳极氧化** | 防腐+装饰 | 铝边框 |
| **喷涂** | 防护+标识 | 外表面 |
| **PVC涂层** | 防石击 | 底板下表面 |

## 典型电池包壳体案例

### 宁德时代CTP电池包

**特点**：
- 大模组设计
- 铝合金边框
- 集成液冷板

**参数**：
- 能量密度：>200Wh/kg

- CTC技术
- 电芯直接排列

**参数**：
- 电芯：4680圆柱
- 框架：与底盘集成
- 底板：一体化压铸

## 发展趋势

1. **轻量化**：铝镁合金、复合材料应用增加
2. **一体化压铸**：底板、边框一体化成型
3. **模块化**：标准化壳体，适配不同电芯
4. **智能化**：集成传感器、自诊断功能
5. **可换电**：快换接口、定位设计
6. **热管理集成**：液冷板与壳体一体化

---

## 总结

电池包壳体是电动车白车身的关键组成部分，其设计直接影响整车的安全性、刚性和轻量化水平。随着CTC/CTB技术的普及，电池包壳体将深度融入车身结构，成为整车承载体系的核心。
