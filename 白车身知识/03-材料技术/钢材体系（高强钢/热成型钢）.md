# 钢材体系

## 概述

钢材是白车身最基础、应用最广泛的材料。经过百年发展，车用钢材已形成完整的等级体系，从普通低碳钢到超高强度热成型钢，满足了不同部位对强度、成型性和成本的要求。

---

## 钢材等级划分

### 按强度等级分类

| 钢材类型 | 屈服强度(MPa) | 抗拉强度(MPa) | 延伸率(%) | 应用部位 |
|----------|---------------|---------------|-----------|----------|
| **普通低碳钢** | ≤210 | ≤340 | ≥35 | 非承力覆盖件 |
| **高强度钢(HSS)** | 210-340 | 340-590 | ≥25 | 地板横梁、门槛外板 |
| **先进高强度钢(AHSS)** | 340-780 | 590-1180 | ≥15 | 前纵梁、A柱内板 |
| **热成型钢(PHS)** | ≥950 | ≥1300 | ≥6 | B柱、车门防撞梁 |

### 钢材类型详解

#### 1. 普通低碳钢（Mild Steel）

**特点**：
- 成本低，成型性好
- 强度低，需要较厚板材
- 焊接性能优异

**应用**：
- 后围板、小尺寸装饰件
- 非承力外覆盖件

**典型牌号**：DC01, DC04, SPCC

#### 2. 高强度钢（High Strength Steel, HSS）

**特点**：
- 强度比普通钢高2-3倍
- 仍保持良好的成型性
- 成本适中

**应用**：
- 地板横梁
- 门槛梁外板
- 顶盖横梁

**典型牌号**：HSLA340, HSLA420

#### 3. 先进高强度钢（Advanced HSS, AHSS）

**子类型**：

| 类型 | 特点 | 应用 |
|------|------|------|
| **双相钢(DP)** | 铁素体+马氏体，高强度+良好成型性 | 前纵梁、地板纵梁 |
| **复相钢(CP)** | 多相组织，高强度+高扩孔性 | 悬挂支架 |
| **相变诱导塑性钢(TRIP)** | 相变吸能，极高延展性 | 碰撞吸能区 |
| **马氏体钢(MS)** | 全马氏体，极高强度 | 局部加强件 |

**典型牌号**：
- DP590, DP780, DP980
- CP800, CP1000
- TRIP780, TRIP980

#### 4. 热成型钢（Press Hardened Steel, PHS）

**工艺原理**：
```
钢板加热(900-950℃) → 高温冲压 → 模具内淬火 → 马氏体组织
```

**性能特点**：
- 屈服强度可达1500MPa以上
- 是普通低碳钢的7倍以上
- 可实现复杂形状成型

**典型牌号**：
- 22MnB5（最常用）
- 34MnB5
- 20MnB8

## 钢材在白车身中的应用分布

### 分区强化设计

```
白车身钢材分布示意图（侧视图）：

顶盖:     高强钢/铝合金
    ┌─────────────────┐
A柱:  ▓▓▓ 热成型钢 ▓▓▓
B柱:  ▓▓▓ 热成型钢 ▓▓▓  ← 最高强度
C柱:  ▓▓▓ 高强钢 ▓▓▓
    ├─────────────────┤
门槛: ▓▓▓ 热成型钢 ▓▓▓  ← 侧碰保护
    └─────────────────┘
地板: 高强钢（电动车+电池加强）
```

### 典型应用部位

| 部位 | 推荐材料 | 强度级别 |
|------|----------|----------|
| **B柱加强板** | 热成型钢 | 1500MPa |
| **A柱加强板** | 热成型钢 | 1000-1500MPa |
| **门槛梁** | 热成型钢/DP钢 | 800-1500MPa |
| **前纵梁** | DP780/DP980 | 590-980MPa |
| **地板横梁** | HSLA340/420 | 340-420MPa |
| **车门防撞梁** | 热成型钢 | 1400-1500MPa |
| **顶盖横梁** | HSLA340 | 340MPa |
| **外覆盖件** | DC04/铝板 | 210MPa以下 |

## 结构优化设计

### 1. 截面优化

**闭口截面 vs 开口截面**：

| 截面类型 | 抗弯刚度 | 抗扭刚度 | 应用 |
|----------|----------|----------|------|
| 开口截面（U型） | 基准 | 基准 | 简单加强件 |
| 闭口截面（O型） | 3-5倍 | 10倍以上 | 纵梁、门槛 |

### 2. 厚度梯度设计

**B柱厚度梯度示例**：
```
上段（支撑车顶）: 1.8mm → 中段（腰部保护）: 2.0mm → 下段（减轻重量）: 1.4mm
```

### 3. 多层板设计

**B柱典型结构**：
```
外层: 0.7mm 深冲钢（外观）
中层: 1.6mm 热成型钢（强度）
内层: 0.8mm 高强钢（结构）
```

## 电动车钢材应用特点

### 1. 更高强度需求

**原因**：
- 电池包重量大，需要更高刚度支撑
- 底部保护需要更强结构
- 一体化设计对连接强度要求高

**趋势**：
- 热成型钢比例从15%提升到25-30%
- 1500MPa级钢材应用增加
- 2000MPa级钢材研发中

### 2. 电池包专用钢材

**吉帕钢（GPa级钢）**：
- 宝钢开发的新能源车专用钢
- 强度1000-1500MPa
- 用于电池包边框、托架

**应用案例**：
- 电池包托架使用QP980（淬火配分钢）
- 减重10%，单车成本降低2.93元

### 3. 连接技术挑战

**高强钢焊接难点**：
- 焊接飞溅大
- 热影响区软化
- 需要优化焊接参数

**解决方案**：
- 自适应焊接系统
- 激光焊接应用
- 胶接-铆接复合连接

## 发展趋势

1. **强度提升**：2000MPa级热成型钢商业化
2. **镀层技术**：Al-Si镀层（Usibor）提高耐蚀性
3. **激光拼焊板（TWB）**：不同厚度/强度钢板焊接后整体冲压
4. **柔性轧制板（TRB）**：连续变厚度钢板
5. **绿色制造**：减少碳排放，氢冶金工艺
