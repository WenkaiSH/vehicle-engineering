# 碳纤维复合材料

## 概述

**碳纤维复合材料（CFRP, Carbon Fiber Reinforced Polymer）** 被誉为"黑色黄金"，其密度仅为钢材的1/5，抗拉强度却是钢材的5倍以上。是白车身轻量化的终极材料选择。

---

## 材料特性

### 基本性能对比

| 性能 | 钢材 | 铝合金 | 碳纤维复合材料 |
|------|------|--------|---------------|
| **密度(g/cm³)** | 7.85 | 2.70 | 1.55-1.60 |
| **抗拉强度(MPa)** | 400-1500 | 100-600 | 1500-3500 |
| **比强度** | 51-191 | 37-222 | **968-2258** |
| **弹性模量(GPa)** | 210 | 70 | 150-300 |
| **比刚度** | 27 | 26 | **97-193** |

### 核心优势

1. **极致轻量化**：比强度和比刚度远超金属
2. **可设计性强**：通过纤维铺层方向优化性能
3. **抗疲劳性好**：疲劳寿命优于金属
4. **耐腐蚀**：不生锈，耐化学腐蚀
5. **功能集成**：电磁屏蔽、导热、与传感器集成

### 局限性

1. **成本高**：原材料价格是钢的10-20倍
2. **成型周期长**：热压罐工艺需要数小时
3. **回收困难**：热固性树脂难以回收
4. **维修复杂**：损伤检测和修复技术要求高
5. **各向异性**：性能与纤维方向相关

## 制造工艺

### 1. 预浸料热压罐工艺（Autoclave）

**流程**：
```
预浸料裁剪 → 铺层 → 真空袋封装 → 热压罐固化(180℃, 0.6MPa) → 脱模 → 修整
```

**优点**：
- 质量高，孔隙率低（<1%）
- 性能稳定

**缺点**：
- 周期长（4-8小时）
- 设备投资大
- 成本高

### 2. 树脂传递模塑（RTM）

**流程**：
```
纤维预制体 → 模具合模 → 树脂注入 → 固化 → 脱模
```

**优点**：
- 周期较短（30-60分钟）
- 可成型复杂形状
- 表面质量好

**应用**：
- 车身覆盖件
- 结构件

### 3. 片状模塑料（SMC）

**特点**：
- 短切纤维增强
- 模压成型
- 生产效率高

**应用**：
- 翼子板
- 发动机罩
- 尾门

### 4. 热塑性复合材料

**特点**：
- 可熔融加工
- 可焊接
- 可回收

**材料**：
- CF/PEEK
- CF/PA6
- CF/PP

## 成本与量产挑战

### 成本构成

| 项目 | 占比 | 说明 |
|------|------|------|
| **原材料** | 40-50% | 碳纤维丝束、树脂 |
| **成型** | 30-40% | 铺层、固化、脱模 |
| **后处理** | 10-20% | 修整、表面处理 |

### 成本对比

| 材料 | 相对成本 | 成型周期 |
|------|----------|----------|
| 钢 | 1x | 1分钟（冲压） |
| 铝 | 3-5x | 1分钟（冲压） |
| CFRP（热压罐） | 10-20x | 4-8小时 |
| CFRP（RTM） | 5-10x | 30-60分钟 |

### 降本路径

1. **原材料降价**：大丝束碳纤维应用
2. **自动化生产**：自动铺丝（AFP）、自动铺带（ATL）
3. **快速固化**：5分钟快速固化树脂
4. **热塑性材料**：可回收，成型快
5. **设计优化**：减少材料用量

---

## 发展趋势

1. **成本下降**：2030年目标降至$10/kg（目前$15-30/kg）
2. **成型提速**：分钟级成型工艺成熟
3. **热塑性应用**：可回收复合材料增长
4. **结构电池**：电池本身成为碳纤维结构的一部分
5. **多材料混合**：CFRP与金属优化组合
6. **中国力量**：国产碳纤维产能扩张，成本下降

---

## 总结

碳纤维复合材料是白车身轻量化的终极解决方案，虽然目前成本较高，但随着技术进步和规模化生产，正在从超跑向高端乘用车、电动车扩展。在电动化时代，碳纤维对提升续航里程的重要性将进一步凸显。
