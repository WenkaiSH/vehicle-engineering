# 前后舱结构

## 概述

**前舱**（发动机舱/前行李舱）和**后舱**（行李舱/后地板）是白车身的重要组成部分，分别承担动力系统布置、碰撞吸能和储物功能。

---

## 前舱结构

### 1. 传统燃油车前舱

**主要部件**：

| 部件 | 功能 | 材料 |
|------|------|------|
| **前围板** | 分隔乘员舱与发动机舱 | 高强钢 |
| **前纵梁** | 主要承载和碰撞吸能结构 | 高强钢/热成型钢 |
| **前减震塔** | 前悬架安装点 | 铝合金/高强钢 |
| **前横梁** | 前保险杠安装，正面碰撞保护 | 高强钢 |
| **翼子板支架** | 翼子板安装支撑 | 低碳钢 |

**碰撞传力路径**：
```
前保险杠 → 前横梁 → 前纵梁 → 地板纵梁 → 乘员舱
                ↓
            吸能盒（压溃变形）
```

### 2. 电动车前舱

**结构变化**：
- 取消发动机，前舱空间释放
- 前行李箱（Frunk）设计
- 前电机布置（四驱车型）

**关键部件**：

| 部件 | 功能 | 特点 |
|------|------|------|
| **前围板** | 分隔乘员舱与前舱 | 加强设计，正面碰撞保护 |
| **前舱储物盒** | 前行李箱 | 需要密封设计 |
| **前电机支架** | 电机安装 | 高刚度要求 |
| **前备箱盖** | 覆盖前行李箱 | 轻量化设计 |

**特斯拉前舱一体化压铸**：
- Model Y前舱采用一体化压铸
- 将多个零件集成为一个总成
- 减少零件数量，提高精度

## 后舱结构

### 1. 传统轿车后舱

**主要部件**：

| 部件 | 功能 | 备注 |
|------|------|------|
| **后地板** | 行李箱地板 | 备胎槽内凹 |
| **后围板** | 后保险杠安装 | 碰撞保护 |
| **后纵梁** | 后部载荷传递 | 追尾碰撞吸能 |
| **轮罩** | 后轮安装空间 | 与侧围连接 |
| **后翼子板** | 车身侧面后部 | 造型覆盖 |

### 2. 电动车后舱

**结构特点**：
- 后电机布置（后驱/四驱）
- 后地板空间优化（无排气管）
- 行李箱空间增大

**特斯拉后地板一体化压铸**：
- Model Y首创后地板一体化压铸
- 原70个零件 → 1个压铸件
- 80个机器人焊点 → 无需焊接
- 制造时间：1-2小时 → 3-5分钟

**一体化压铸优势**：
1. 零件数量大幅减少
2. 尺寸精度提高
3. 重量减轻
4. 成本降低（量产阶段）

## 电动车前后舱一体化趋势

### 前后舱一体化压铸方案

```
传统方案：                    一体化压铸方案：
┌─────────┐                  ┌─────────┐
│ 多个冲压件 │                  │ 单一压铸件 │
│ + 焊接   │        →         │ 直接成型  │
│ = 前舱总成 │                  │ = 前舱总成 │
└─────────┘                  └─────────┘
```

## 材料应用

### 传统方案

- **前纵梁**：高强钢（340-590MPa）+ 热成型钢加强
- **前围板**：高强钢多层板设计
- **后地板**：低碳钢，厚度0.7-1.0mm

### 一体化压铸方案

- **材料**：免热处理铝合金（如特斯拉自研合金）
- **工艺**：高压压铸（HPDC）
- **优点**：无需热处理，减少变形

---

## 性能要求

### 正面碰撞（前舱）

| 测试项目 | 要求 |
|----------|------|
| 100%正面碰撞 | 乘员舱完整性，假人伤害值达标 |
| 40%偏置碰撞 | 纵梁有效压溃吸能 |
| 行人保护 | 发动机罩变形吸能 |

### 追尾碰撞（后舱）

| 测试项目 | 要求 |
|----------|------|
| 高速追尾 | 燃油系统完整性（燃油车） |
| 低速碰撞 | 可维修性，低成本修复 |

---

## 发展趋势

1. **一体化压铸普及**：从后地板扩展到前舱、侧围
2. **材料创新**：免热处理合金、高强韧铝合金
3. **结构集成**：前舱+前围一体化设计
4. **功能集成**：预埋线束、传感器支架
