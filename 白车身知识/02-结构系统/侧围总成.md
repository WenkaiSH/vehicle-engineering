# 侧围总成

## 概述

**侧围总成**是白车身两侧的大型覆盖结构，由外板、内板及加强件组成，是车身侧面造型和结构强度的关键部分。

---

## 组成结构

### 1. 侧围外板

**功能**：
- 构成车身侧面外观
- 覆盖车门开口
- 与翼子板、顶盖衔接

**特点**：
- 面积大（单件可达2-3㎡）
- 形状复杂（多曲面）
- 表面质量要求高

### 2. 侧围内板

包含以下主要结构：

| 结构件 | 位置 | 功能 |
|--------|------|------|
| **A柱内板** | 前风挡两侧 | 支撑前风挡，传递正面碰撞力 |
| **B柱内板** | 前后门之间 | 侧面碰撞保护，车顶支撑 |
| **C柱内板** | 后门与后窗之间 | 支撑后窗，传递后部载荷 |
| **门槛内板** | 底部 | 侧面碰撞保护，电池防护（电动车） |
| **上边梁** | 顶部 | 顶盖支撑，翻滚保护 |

### 3. 加强件系统

- **A柱加强板**：多层板叠焊，热成型钢
- **B柱加强板**：最高强度区域，热成型钢1500MPa+
- **门槛加强梁**：闭口截面设计，侧面碰撞吸能
- **窗框加强**：保证侧围刚度

---

## 电动车侧围的特殊设计

### 1. 门槛梁加强

**传统燃油车门槛**：
- 功能：侧面碰撞保护
- 材料：高强钢
- 截面：单腔或双腔

**电动车门槛**：
- 功能：侧面碰撞保护 + **电池防护**
- 材料：热成型钢/铝合金复合
- 截面：多腔体设计，填充蜂窝铝或泡沫

### 2. B柱区域优化

电动车B柱设计特点：
- 底部与门槛加强梁连接
- 中部避开电池包位置
- 顶部与顶盖纵梁连接
- 形成完整的侧面保护环

### 3. 材料分布策略

```
侧围材料分布（从外到内）：
┌─────────────────────────────────┐
│  外板：低碳钢/铝合金（表面质量）  │
├─────────────────────────────────┤
│  加强板：高强钢/热成型钢（强度）  │
├─────────────────────────────────┤
│  内板：高强钢（结构支撑）        │
└─────────────────────────────────┘
```

## 制造工艺

### 冲压工艺

**侧围外板**：
- 尺寸：约2.5m × 1.5m
- 材料：DC04（深冲用钢）
- 工序：拉延→修边→冲孔→翻边
- 设备：6000吨级压力机

### 焊接工艺

**侧围总成焊点分布**：
- A柱区域：80-120点
- B柱区域：60-100点
- C柱区域：40-60点
- 门槛区域：100-150点

**连接方式**：
- 点焊：主体连接
- 激光焊：顶盖连接
- SPR：铝合金部件

## 性能要求

### 侧面碰撞性能

| 测试项目 | 要求 | 评价指标 |
|----------|------|----------|
| 可变形移动壁障 | 侵入量 ≤ 150mm | 乘员舱完整性 |
| 柱碰试验 | 侵入量 ≤ 120mm | 电池安全（电动车） |
| 侧面气囊展开 | 不得撕裂 | 材料韧性 |

### 刚度要求

- **侧围整体刚度**：与车身扭转刚度匹配
- **窗框刚度**：防止车门玻璃异常振动
- **B柱刚度**：支撑车门铰链和锁扣

---

## 发展趋势

1. **高强度化**：热成型钢比例提升至30%+
2. **轻量化**：铝合金侧围应用（奥迪A8、特斯拉Model S）
3. **模块化**：侧围分总成预制，主线快速装配
4. **一体化压铸**：后侧围一体化压铸（特斯拉Model Y）
