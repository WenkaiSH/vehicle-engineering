# 地板总成

## 概述

**地板总成**是白车身的基础承载结构，承担垂直载荷与横向载荷的传递。它是车身骨架的核心，直接影响整车的刚度、强度和NVH性能。

---

## 主要组成部件

### 1. 地板面板

| 部件 | 位置 | 功能 |
|------|------|------|
| **前地板** | 乘员舱前部 | 支撑前排座椅，安装制动/转向系统 |
| **中地板** | 乘员舱核心区域 | 设置座椅安装点、安全带固定点 |
| **后地板** | 乘员舱后部 | 包含备胎槽或货箱安装结构 |

### 2. 加强梁系

- **地板纵梁**：沿车身长度方向布置，是碰撞能量传递的主要路径
- **地板横梁**：横向布置，连接左右纵梁，提升抗扭性能
- **座椅横梁**：专门承载座椅安装载荷
- **中央通道**：传动轴通道（燃油车）/高压线束通道（电动车）

### 3. 功能附件

- **备胎槽**：轿车后地板内凹形成的储物空间
- **油箱安装支架**：燃油车燃油箱固定点
- **线束/管路通道**：电气和流体系统的布置空间

---

## 电动车地板总成的特殊性

### 1. 电池包集成设计

**传统燃油车地板**：
```
地板面板 → 中央通道 → 排气管/传动轴
```

**电动车地板**：
```
地板面板 → 电池包 → 纯平或微凸设计
```

### 2. 结构形式演变

| 类型 | 特点 | 代表方案 |
|------|------|----------|
| **传统分体式** | 电池包独立安装于车身下方 | 早期电动车 |
| **CTB集成式** | 电池包上盖兼任车身地板 | 比亚迪e平台3.0 |
| **CTC滑板式** | 电池与底盘一体化 | 特斯拉/宁德时代 |

### 3. 电动车地板设计要点

1. **纯平化**：最大化电池布置空间
2. **侧向保护**：门槛梁与电池包边框集成加强
3. **底部防护**：下护板+防护涂层
4. **密封性**：IP67级防水防尘要求

## 材料应用

### 传统方案

- **材料**：低碳钢、高强度钢
- **厚度**：0.6-1.2mm
- **工艺**：冲压成型+点焊

### 轻量化方案

- **材料**：铝合金板材、碳纤维复合材料
- **连接**：SPR自冲铆接+结构胶
- **效果**：减重20-30%

### 电动车优化方案

- **电池包上盖**：高强钢或铝合金（兼任地板功能）
- **边框结构**：热成型钢（侧碰保护）
- **底部护板**：铝合金或复合材料

---

## 制造工艺

### 主要工序

1. **冲压成型**：各零件单独冲压
2. **分总成焊接**：小总成预制
3. **地板总成焊接**：机器人自动焊接
4. **精度检测**：三坐标测量

### 关键工艺参数

- **焊点密度**：20-40点/㎡
- **尺寸精度**：±0.5mm（关键定位点）
- **平面度**：≤2mm/m²

---

## 性能要求

| 性能指标 | 目标值 | 测试方法 |
|----------|--------|----------|
| 弯曲刚度 | ≥20,000 N/mm | 三点弯曲试验 |
| 模态频率 | ≥35 Hz（一阶弯曲） | 模态分析 |
| 耐久性 | 20万公里无裂纹 | 道路模拟试验 |

---

## 发展趋势

1. **模块化设计**：地板总成与电池包模块化供货
2. **一体化压铸**：后地板一体化压铸减少零件数量
3. **材料升级**：钢-铝混合结构普及
4. **智能化**：集成传感器、线束预埋
