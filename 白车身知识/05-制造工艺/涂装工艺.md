# 涂装工艺

## 概述

**涂装工艺**是白车身制造的第三道大工序，为车身提供防腐保护、装饰外观和特殊功能（如隔热、降噪）。涂装质量直接影响车辆的耐久性和美观度。

---

## 涂装工艺流程

### 完整工艺流程

```
白车身 → 前处理 → 电泳 → 密封 → 中涂 → 色漆 → 清漆 → 检查 → 总装
         ↓        ↓      ↓      ↓      ↓      ↓
       脱脂     底漆   涂胶   功能层  颜色   保护层
       表调     防腐   密封   填充   美观   光泽
       磷化
       钝化
```

---

## 前处理

### 目的

去除车身表面油污、氧化皮，形成转化膜，提高涂层附着力。

### 工序

| 工序 | 功能 | 材料/方法 |
|------|------|----------|
| **预清洗** | 去除大颗粒污物 | 高压水喷淋 |
| **预脱脂** | 去除大部分油脂 | 碱性脱脂剂 |
| **主脱脂** | 彻底除油 | 碱性脱脂剂+喷淋 |
| **水洗** | 清洗残留 | 纯水喷淋 |
| **表调** | 调整表面状态 | 表调剂 |
| **磷化** | 形成磷化膜 | 锌系/锰系磷化液 |
| **水洗** | 清洗残留 | 纯水喷淋 |
| **钝化** | 提高耐蚀性 | 无铬钝化剂 |
| **纯水洗** | 最终清洗 | 去离子水 |

### 磷化膜

**作用**：
- 提高涂层附着力
- 提高防腐性能
- 改善电泳涂装性能

**参数**：
- 膜重：1.5-3.0 g/m²
- 厚度：3-5 μm

## 电泳涂装（ED）

### 原理

```
车身（阳极） → 电泳槽液（含树脂颜料） → 电源 → 电沉积
                 ↑
            电场作用下，带电粒子沉积于车身表面
```

### 工艺特点

| 特点 | 说明 |
|------|------|
| **全覆盖** | 内腔、缝隙都能覆盖 |
| **自动化** | 100%自动化生产 |
| **环保** | 水性涂料，VOC低 |
| **高效** | 涂料利用率高（95%+） |

### 工艺参数

| 参数 | 范围 |
|------|------|
| **电压** | 200-400 V |
| **时间** | 3-5 min |
| **温度** | 28-32℃ |
| **膜厚** | 18-25 μm |

### 烘烤固化

- 温度：160-180℃
- 时间：20-30 min
- 膜层：交联固化，形成致密涂层

## 密封涂胶

### 目的

- 密封车身缝隙，防止水汽、灰尘进入
- 隔振降噪
- 增强车身刚度

### 密封部位

| 部位 | 作用 |
|------|------|
| **焊缝密封** | 防水防锈 |
| **地板涂胶** | 防石击、隔音 |
| **车身折边** | 密封、防锈 |
| **前后围** | 密封、隔音 |

### 密封材料

| 类型 | 特点 | 应用 |
|------|------|------|
| **PVC密封胶** | 弹性好，成本低 | 焊缝密封 |
| **折边胶** | 强度高 | 车门折边 |
| **膨胀胶** | 烘烤后膨胀 | 空腔填充 |
| **隔振胶** | 减振降噪 | 地板 |

## 中涂涂装

### 目的

- 提高面漆附着力
- 填充微小缺陷
- 提高抗石击性
- UV防护

### 工艺

| 项目 | 参数 |
|------|------|
| **喷涂方式** | 机器人静电喷涂 |
| **膜厚** | 30-40 μm |
| **烘烤** | 140-150℃ × 20min |

### 打磨

- 目的：消除颗粒、流挂
- 方式：手工+机器人
- 材料：P400-P600砂纸

## 面漆涂装

### 色漆（Base Coat）

**作用**：提供颜色

**类型**：
| 类型 | 特点 | 应用 |
|------|------|------|
| **水性色漆** | 环保，VOC低 | 主流 |
| **溶剂型色漆** | 光泽好 | 高性能车 |
| **珠光漆** | 金属光泽 | 豪华车 |
| **幻彩漆** | 随角度变色 | 个性车 |

**工艺**：
- 喷涂：2-3层
- 膜厚：15-25 μm
- 闪干：2-5 min

### 清漆（Clear Coat）

**作用**：
- 保护色漆
- 提供光泽
- UV防护
- 耐划伤

**类型**：
| 类型 | 特点 |
|------|------|
| **溶剂型清漆** | 光泽好，硬度高 |
| **水性清漆** | 环保 |
| **高固分清漆** | 少用溶剂 |
| **UV清漆** | 快速固化 |

**工艺**：
- 喷涂：1-2层
- 膜厚：40-60 μm
- 烘烤：140-150℃ × 30min

## 涂装质量控制

### 外观质量

| 缺陷 | 原因 | 预防 |
|------|------|------|
| **颗粒** | 环境污染 | 洁净室 |
| **流挂** | 涂料过厚 | 控制膜厚 |
| **缩孔** | 油污污染 | 前处理 |
| **色差** | 涂料批次 | 严格配色 |
| **桔皮** | 流平不好 | 优化工艺 |

### 性能测试

| 测试 | 方法 | 要求 |
|------|------|------|
| **附着力** | 划格法 | 0-1级 |
| **硬度** | 铅笔硬度 | ≥H |
| **耐冲击** | 落锤冲击 | ≥50 cm·kg |
| **耐盐雾** | 盐雾试验 | 720h无起泡 |
| **耐候性** | 紫外老化 | 2000h无变色 |

## 发展趋势

1. **水性化**：全面水性涂料
2. **高固体分**：减少溶剂使用
3. **粉末涂装**：零VOC
4. **数字化**：智能涂装，质量预测
5. **免中涂工艺**：简化流程，节能环保

---

## 总结

涂装工艺是白车身制造的关键环节，通过多层涂层体系为车身提供防腐、美观和功能保护。随着环保要求提高，水性化、高固体分和粉末涂装是发展方向。
