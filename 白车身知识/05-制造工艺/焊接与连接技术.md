# 焊接与连接技术

## 概述

**焊接与连接技术**是将数百个冲压零件组装成完整白车身的核心工艺。现代白车身制造采用"点焊为主，多种连接方式并存"的混合连接策略，以满足不同材料和结构的连接需求。

---

## 点焊（Resistance Spot Welding）

### 原理

```
电极加压 → 通电加热 → 熔核形成 → 冷却结晶 → 焊点形成
   ↓           ↓          ↓          ↓
 压力      电阻热      熔化金属    固态连接
```

### 工艺参数

| 参数 | 范围 | 说明 |
|------|------|------|
| **焊接电流** | 8-20 kA | 决定热量输入 |
| **焊接时间** | 0.1-0.5 s | 单点焊接周期 |
| **电极压力** | 2-5 kN | 保证接触导电 |
| **电极头直径** | 5-8 mm | 影响焊核大小 |

### 焊点质量

**合格标准**：
- 焊核直径：≥4√t（t为板厚）
- 压痕深度：≤20%板厚
- 无飞溅、无裂纹、无气孔

**检测方法**：
- 凿检（破坏性）
- 超声波检测（无损）
- 目视检查

### 应用

- 占白车身焊点总数80%以上
- 主要用于钢板连接
- 不适合铝-钢连接

---

## 激光焊接（Laser Welding）

### 激光熔焊

**原理**：高能量密度激光束熔化材料，形成焊缝。

**特点**：
- 能量密度高（10⁶ W/cm²）
- 焊接速度快（1-10 m/min）
- 热影响区小
- 变形小

**应用**：
- 车顶与侧围连接
- 后盖焊接
- 地板焊缝

### 激光钎焊

**原理**：激光加热母材，熔化钎料填充间隙。

**特点**：
- 焊缝美观（无需打磨）
- 连接异种材料
- 强度略低于熔焊

**应用**：
- 顶盖与侧围（外观质量要求高）
- 铝-钢连接

### 激光焊接参数

| 参数 | 范围 |
|------|------|
| **激光功率** | 2-10 kW |
| **焊接速度** | 1-8 m/min |
| **离焦量** | ±2 mm |
| **保护气体** | Ar, He |

---

## 弧焊（Arc Welding）

### MIG/MAG焊

**原理**：熔化极惰性/活性气体保护焊。

**应用**：
- 厚板结构件
- 补焊
- 小批量生产

### 特点

- 设备简单，灵活
- 热输入大，变形大
- 需要后续打磨

---

## 异种材料连接技术

### 钢-铝连接挑战

| 物理性质 | 钢 | 铝 | 影响 |
|----------|-----|-----|------|
| 熔点 | ~1538℃ | ~660℃ | 无法直接点焊 |
| 热导率 | 50 W/m·K | 237 W/m·K | 散热快 |
| 线膨胀系数 | 12×10⁻⁶/K | 23×10⁻⁶/K | 热变形差异大 |

### 自冲铆接（SPR）

**原理**：
```
半空心铆钉 → 穿透上层板材 → 在下层扩张 → 机械锁合
```

**特点**：
- 可连接钢-铝、铝-铝
- 强度高，疲劳性能好
- 无热影响

**应用**：
- 奥迪全铝车身
- 电动车电池包
- 钢铝混合车身

**设备**：C型或弓型铆接机

### 流钻螺钉（FDS）

**原理**：
```
高速旋转（2000-5000 rpm）→ 摩擦生热 → 材料软化 → 螺钉攻入
```

**特点**：
- 单面可达
- 可拆卸
- 适合狭小空间

**应用**：
- 电池包安装
- 内部连接
- 维修位置

### 搅拌摩擦焊（FSW）

**原理**：
```
搅拌头高速旋转 → 摩擦生热 → 材料塑化 → 固相连接
```

**特点**：
- 无熔化缺陷
- 焊接质量高
- 变形小
- 适合铝合金

**应用**：
- 电池包底板
- 铝合金型材
- 长焊缝

## 粘接技术

### 结构胶粘接

**材料**：
- 环氧树脂胶
- 聚氨酯胶
- 丙烯酸胶

**特点**：
- 面连接，应力分布均匀
- 提高刚度20-30%
- 密封、减振
- 耐腐蚀

**应用**：
- 外覆盖件粘接
- 钢铝混合连接（辅助）
- 密封补强

### 粘接+机械连接

**组合方式**：
- 粘接+点焊
- 粘接+SPR
- 粘接+FDS

**优势**：
- 刚度提升30-50%
- 疲劳性能改善
- 密封可靠

## 其他连接技术

### 螺栓连接

**应用**：
- 可拆卸部位
- 维修位置
- 附件安装

**特点**：
- 可拆卸
- 需防松措施
- 成本高

### 卡扣连接

**应用**：
- 内饰件
- 外饰件
- 线束固定

### 一体化压铸连接

**特斯拉创新**：
- 压铸件与车身焊接
- 减少连接点
- 提高精度

## 连接方式选择指南

| 材料组合 | 推荐连接方式 | 备注 |
|----------|-------------|------|
| **钢-钢** | 点焊 | 主流 |
| **铝-铝** | 激光焊、SPR | 质量高 |
| **钢-铝** | SPR + 结构胶 | 异种材料 |
| **钢-碳纤维** | 结构胶 + 机械连接 | 绝缘处理 |
| **铝-碳纤维** | 结构胶 + 铆接 | 热膨胀差异 |
| **塑料-金属** | 卡扣 + 粘接 | 非承力 |

## 连接质量检测

### 在线检测

| 方法 | 原理 | 应用 |
|------|------|------|
| **超声波检测** | 声波反射 | 点焊、焊缝 |
| **X射线检测** | 穿透成像 | 焊缝内部 |
| **涡流检测** | 电磁感应 | 表面缺陷 |

### 破坏性检测

- 金相分析
- 拉伸试验
- 剥离试验
- 疲劳试验

## 发展趋势

1. **连接技术多样化**：适应多材料车身
2. **自动化检测**：100%在线检测
3. **数字化监控**：实时质量监控
4. **新型连接工艺**：电磁脉冲焊接、摩擦焊
5. **仿真优化**：连接工艺虚拟验证

---

## 总结

焊接与连接技术是白车身制造的核心，从传统的点焊发展到现在的多种连接方式并存，适应了钢铝混合车身和电动车的制造需求。合理选择连接方式对保证白车身质量至关重要。
