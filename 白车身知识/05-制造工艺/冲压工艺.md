# 冲压工艺

## 概述

**冲压工艺**是白车身制造的第一道工序，通过模具对金属板材施加压力，使其产生塑性变形，获得所需形状和尺寸的零件。冲压质量直接决定白车身的尺寸精度和表面质量。

---

## 冲压工艺原理

### 基本工序

| 工序 | 功能 | 变形特点 |
|------|------|----------|
| **落料** | 将板材切割成所需形状的毛坯 | 分离变形 |
| **拉延** | 将平板坯料成型为立体零件 | 拉伸+压缩 |
| **修边** | 切除多余材料 | 分离变形 |
| **冲孔** | 在零件上冲出孔洞 | 分离变形 |
| **翻边** | 将边缘折弯 | 弯曲变形 |
| **整形** | 修正形状和尺寸 | 小变形校正 |

### 拉延成型原理

**变形过程**：
```
平板坯料 → 凸模下压 → 材料流动 → 贴模成型
                ↓
        凸缘材料流入凹模（补充拉伸区）
```

**关键参数**：
- 压边力：控制材料流动
- 拉延筋：调节进料阻力
- 模具间隙：1.0-1.2倍料厚

## 冲压设备

### 压力机类型

| 类型 | 特点 | 应用 |
|------|------|------|
| **机械压力机** | 速度快，刚性高 | 中小件批量生产 |
| **液压压力机** | 压力大，速度可控 | 大型覆盖件 |
| **伺服压力机** | 滑块运动可编程 | 精密成型 |
| **多工位压力机** | 多工序连续 | 自动化生产线 |

### 压力机吨位

| 零件类型 | 吨位范围 | 典型应用 |
|----------|----------|----------|
| **小件** | 200-800吨 | 支架、加强板 |
| **中件** | 1000-2000吨 | 车门内板、地板 |
| **大件** | 3000-6000吨 | 侧围外板、机盖外板 |

## 白车身典型零件冲压

### 1. 覆盖件冲压

#### 侧围外板

**特点**：
- 尺寸大（约2.5m × 1.5m）
- 形状复杂（多曲面）
- 表面质量要求高（A级表面）

**工序**：
```
落料 → 拉延 → 修边冲孔 → 翻边整形 → 检验
```

**关键控制**：
- 拉延深度：均匀分布
- 材料流动：防止起皱和破裂
- 回弹控制：模具补偿

#### 发动机罩外板

**特点**：
- 面积大而浅
- 曲率变化平缓
- 外表面要求无缺陷

**工艺难点**：
- 刚性不足（需要加强筋）
- 回弹大
- 外观质量要求高

### 2. 结构件冲压

#### 纵梁

**特点**：
- 高强钢材料
- U型截面
- 孔位精度要求高

**工序**：
```
落料 → 拉延/成型 → 修边冲孔 → 翻边 → 侧整形
```

#### 地板零件

**特点**：
- 平板+局部凸起
- 多孔位（安装点）
- 尺寸精度要求高

### 3. 高强钢冲压

#### 挑战

| 问题 | 原因 | 解决措施 |
|------|------|----------|
| **回弹大** | 材料强度高 | 模具补偿，整形工序 |
| **开裂** | 延伸率低 | 优化拉延筋，增加压边力 |
| **模具磨损** | 材料硬 | 模具材料升级，涂层 |

#### 热成型工艺

**流程**：
```
钢板加热（900-950℃）→ 高温冲压 → 模具内淬火 → 马氏体组织
```

**优点**：
- 成型性极好（高温下）
- 强度极高（1500MPa+）
- 回弹极小

**应用**：
- B柱加强板
- 门槛加强板
- 防撞梁

## 冲压质量控制

### 1. 尺寸精度

| 精度等级 | 公差范围 | 应用 |
|----------|----------|------|
| **普通级** | ±0.5mm | 隐藏部位 |
| **精密级** | ±0.3mm | 配合部位 |
| **超精密级** | ±0.1mm | 关键定位点 |

### 2. 表面质量

| 缺陷类型 | 原因 | 预防措施 |
|----------|------|----------|
| **皱纹** | 材料流动不均 | 优化压边力，拉延筋设计 |
| **破裂** | 过度拉伸 | 增加材料流入，减少变形 |
| **划痕** | 模具表面粗糙 | 模具抛光，清洁 |
| **回弹** | 弹性恢复 | 模具补偿，整形 |

### 3. 检测方法

- **三坐标测量**：关键尺寸
- **蓝光扫描**：全型面检测
- **目视检查**：表面缺陷
- **检具匹配**：装配验证

## 发展趋势

1. **伺服压力机普及**：运动曲线可编程，成型性更好
2. **热成型扩展**：更多零件采用热成型
3. **铝材冲压**：铝合金覆盖件增加
4. **数字化**：虚拟调试，数字孪生
5. **柔性生产**：快速换模，多品种小批量

---

## 总结

冲压工艺是白车身制造的基础，通过合理的模具设计和工艺参数控制，可以获得高质量的冲压零件。热成型技术的应用使高强钢零件的成型成为可能，显著提升了白车身的安全性能。
