# 比亚迪CTB技术

## 背景

**CTB（Cell to Body）电池车身一体化技术**是比亚迪e平台3.0的核心创新，首次应用于海豹车型。通过将刀片电池与车身深度融合，实现了安全性、空间利用率和整车性能的全面突破。

---

## CTB技术原理

### 传统方案 vs CTB方案

**传统电池布置**：
```
乘员舱地板（车身）
        ↓ 间隙（浪费空间）
电池包上盖（独立）
        ↓
模组框架
        ↓
电芯（模组内）
        ↓
电池包底板
```

**CTB方案**：
```
电池包上盖（兼任乘员舱地板） ← 一体化设计
        ↓
刀片电芯（无模组，直接排列）
        ↓
电池包底板（与边框集成）
        ↓
车身纵梁（与电池包边框一体化）
```

### 核心技术特点

| 技术 | 说明 | 效果 |
|------|------|------|
| **刀片电池** | 磷酸铁锂，刀片形状，直接集成 | 高安全，高空间利用率 |
| **电池上盖=地板** | 取消传统车身地板 | 空间+10mm |
| **边框一体化** | 电池包边框与车身纵梁连接 | 刚度提升 |
| **蜂窝铝底板** | 底板结构加强 | 底部防护 |

---

## 刀片电池技术

### 结构创新

**传统电池**：
- 电芯 → 模组 → 电池包
- 三层结构，空间利用率低

**刀片电池（CTP）**：
- 电芯直接集成到电池包
- 取消模组
- 空间利用率提升50%

```
传统模组：                      刀片电池：

┌─────────────────┐            ┌─────────────────┐
│  模组外壳        │            │                 │
│  ┌───┬───┬───┐  │            │  刀片电芯直立   │
│  │电芯│电芯│电芯│  │     →    │  ┌───┐          │
│  └───┴───┴───┘  │            │  │   │          │
└─────────────────┘            │  └───┘          │
                                │     ...         │
                                └─────────────────┘
```

### 刀片电芯特点

| 参数 | 数值 |
|------|------|
| **长度** | 900-2500mm（可定制） |
| **厚度** | 约13.5mm |
| **高度** | 约90mm |
| **材料** | 磷酸铁锂 |
| **形状** | 长条形薄板 |

### 安全优势

**针刺试验**：
- 传统三元锂电池：起火爆炸
- 刀片电池：无明火，表面温度仅30-60℃

**原因**：
- 磷酸铁锂材料本身安全
- 大尺寸散热好
- 结构强度高

## CTB结构详解

### 上盖设计

**功能兼任**：
- 电池包密封上盖
- 乘员舱地板
- 座椅安装基础

**材料**：高强钢
**厚度**：1.0-1.2mm
**加强**：横梁集成

### 电芯布置

**竖直布置**：
```
车身横向：

门槛梁 ─┬─ 刀片电芯（竖直）─┬─ 门槛梁
        │                    │
        │  刀片电芯（竖直）   │
        │                    │
        │  刀片电芯（竖直）   │
        │                    │
门槛梁 ─┴─ 刀片电芯（竖直）─┴─ 门槛梁
```

**固定方式**：
- 电芯底部粘接
- 侧面结构胶
- 顶部压板

### 底板与边框

**边框**：
- 高强钢辊压成型
- 与车身门槛梁连接
- 侧碰保护

**底板**：
- 蜂窝铝结构
- 底部防护
- 轻量化

## CTB性能表现

### 扭转刚度

| 车型 | 扭转刚度 | 说明 |
|------|----------|------|
| 传统中型轿车 | 25,000 N·m/deg | 基准 |
| 比亚迪海豹（CTB） | 40,500 N·m/deg | **+62%** |
| 豪华车（燃油车） | 30,000-35,000 | 参考 |

**提升原因**：
- 电池包参与承载
- 电池上盖与车身一体化
- 边框与车身纵梁连接

### 侧面碰撞安全

**测试结果**：
- 侧碰侵入量：减少50%
- 乘员舱完整性：优秀

**保护机制**：
```
侧向冲击 → 门槛梁 → 电池包边框 → 电芯阵列 → 分散载荷
              ↓           ↓
         高强钢边框    电芯刚性强
```

### 空间优化

| 指标 | 提升 |
|------|------|
| **垂向空间** | +10mm（地板降低） |
| **得房率** | 提升（空间利用率） |
| **乘坐空间** | 改善 |

### 续航提升

**因素**：
- 空间利用率提升 → 可装更多电池
- 整车轻量化 → 能耗降低
- 磷酸铁锂成本低 → 同价位更多电量

## CTB vs 竞品技术

| 技术 | 代表 | 特点 | 与CTB对比 |
|------|------|------|-----------|
| **CTC** | 特斯拉 | 电芯直接到底盘 | CTB保留电池包结构，维修性好 |
| **CTP** | 宁德时代 | 大模组/无模组 | CTB是CTP的延伸，集成到车身 |
| **弹匣电池** | 广汽 | 安全系统 | CTB侧重结构集成 |
| **魔方电池** | 上汽 | 换电设计 | CTB侧重一体化 |
