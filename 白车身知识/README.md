# 白车身知识库

## 概述

本知识库系统梳理了**汽车白车身（Body in White, BIW）**的专业知识，涵盖基础概念、结构系统、材料技术、电动车核心技术、制造工艺、设计方法与验证、行业案例等七大模块，共30篇文档。

特别聚焦于**电动车白车身**的技术革新，包括CTC/CTB/CIB电池车身一体化技术、一体化压铸工艺、电动车碰撞安全等前沿内容。

## 核心知识点速览

### 白车身定义
- **定义**：完成焊接但尚未涂装的车身总成
- **地位**：被誉为"汽车之骨"，占整车开发周期40-60%
- **功能**：承载、保护、安装基础

### 电动车白车身特点
- **载荷变化**：电池包重量占整车30-40%
- **刚度提升**：扭转刚度目标从25,000提升到35,000+ N·m/deg
- **电池保护**：碰撞设计核心是保护电池

### 电池车身一体化技术

| 技术 | 代表企业 | 集成度 | 特点 |
|------|----------|--------|------|
| **CTC** | 特斯拉 | 最高 | 电芯直接到底盘 |
| **CTB** | 比亚迪 | 高 | 电池上盖兼任地板 |
| **CIB** | 华为/小米 | 中高 | 电池融入车身结构 |

### 一体化压铸
- **效果**：零件数量减少90%+，制造时间缩短95%
- **应用**：特斯拉Model Y后地板70个零件→1个
- **压铸机**：6000T→9000T→12000T演进

### 材料体系
- **钢材**：热成型钢1500MPa+，用于B柱等关键部位
- **铝合金**：密度钢的1/3，用于覆盖件和电池包
- **碳纤维**：比强度是钢10倍，高端应用
